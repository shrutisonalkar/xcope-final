# XCOPE - AI-Powered Sentiment Analytics Dashboard

## Tech Stack
- **Backend**: Flask, SocketIO, TextBlob (NLP)
- **Frontend**: Tailwind CSS, Chart.js, Vanilla JS
- **Data**: Real-time streaming, Big Data CSV export (10k+ records)

## Features
- Live sentiment analysis on streaming tweets
- Interactive dashboard with charts & stats
- Advanced search/pagination/filtering
- Responsive professional UI
- Production-ready deployment

## Run Locally
1. `pip install -r requirements.txt`
2. `python app.py`
3. Visit http://localhost:5000
